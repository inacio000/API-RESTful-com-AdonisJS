# AdonisJS
## O que é?
*   Um `framework Node.js`, para desenvolver aplicações web;
*   Facilita muito a programação de apps, possui uma estrutura similar ao `Laravel`;
*   Utiliza arquitetura `MVC`;
*   Possui vários recursos, como: `CLI, File upload` simples, `validação` e etc;
*   Há também outros pacotes externos para complementar o ecossistema, como `ORM, Autenticação, Autorização`;
#
## O que vamos criar?
*   Criaremos uma `API RESTful`;
*   Ela terá `CRUD` e relacionamento entre entidades;
*   Outro foco será aprender a utilizar a `CLI do Adonis e outros recursos` exclusivos do framework;
*   Faremos os `testes via Postman`, para garantir o correto funcionamento da API;
*   Futuramente pode se implementar um `front-end para este projeto`;
#
## Pré-requisitos
*   Conhecimento em desenvolvimento web `(app e APIs)`;
*   Ter tido contato com `JavaScript & Node.js`;
*   Desejável: `MVC` e ter criado algum projeto com outro framework, como `Express`;
#
## O que vamos precisar?
*   Ter o `Node.js` instalado na máquina, para instalação de pacotes e execução de scripts;
*   Algum editor de código, sugestão: `VS Code`;
*   Software para teste de APIs, sugestão: `Postman`;
